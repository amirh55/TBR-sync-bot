# 🔄 TBR Sync Bot (Telegram - Bale - Rubika)

> ⚠️ **توجه:**  
> این ربات فقط برای استفاده شخصی یا تیمی مجاز است. قبل از انتشار، قوانین هر پلتفرم را رعایت کنید.

---

  دربارهٔ پروژه
 یک ربات دوطرفه (بله ↔ تلگرام) است که پیام‌ها و فایل‌های ارسال‌شده در کانال بله را به‌صورت لحظه‌ای به کانال تلگرام منتقل می‌کند.  
- **زبان:** پایتون ۳.۱۰+  
- **کتابخانه‌ها:** `python-bale-bot`, `python-telegram-bot` و چند ابزار کمکی (dotenv, aiohttp …)

---


---

### پیش‌نیازها

| مورد | توضیح |
|------|-------|
| **Python** | نسخه‌ی ۳.۱۰ یا بالاتر (شما می‌توانید از `pyenv` یا `conda` استفاده کنید) |
| **Token های ربات‌ها** | توکن بله، تلگرام و (اختیاری) روبیکا |
| **کانال‌ها** | شناسه کانال‌های هدف (`@channel_name`) |

> ⚡️ برای گرفتن توکن‌های ربات:  
> - **Telegram:** با @BotFather یک ربات بسازید و توکن را دریافت کنید.  
> - **Bale:** با @BotFather یک ربات بسازید و توکن را دریافت کنید.

### ۱. کلون کردن پروژه و ورود به پوشه

```bash
# کلون کردن مخزن
git clone https://github.com/amirh55/TBR-sync-bot.git
cd TBR-sync-bot
```

### ۲. ساخت محیط مجازی (Virtual Environment)

```bash
python3 -m venv venv
source venv/bin/activate
```
# نصب وابستگی‌ها
```bash
pip install -r requirements.txt
```
### ۳. اجرای اسکریپت راه‌انداز هوشمند

این دستور تمام کتابخانه‌های مورد نیاز را نصب کرده و سپس توکن‌ها و آیدی کانال‌ها را به صورت تعاملی از شما می‌پرسد و فایل تنظیمات را می‌سازد:
```bash
python3 setup.py
```
بعد از ستاپ برای اجرای فایل به صورت دستی از کامند زیر میتوانید استفاده کنید:
```bash
python main.py
```

🛠️ روشن نگه‌داشتن ربات پس از بستن ترمینال (بسیار مهم)

### ۱. نصب ابزار PM2 روی سرور (فقط یک‌بار)
```bash
sudo apt update && sudo apt install nodejs npm -y
sudo npm install pm2 -g
```

### ۲. روشن کردن ربات در پس‌زمینه با یک نام دلخواه (مثلاً tbr-bot)
```bash
pm2 start venv/bin/python --name "tbr-bot" -- main.py
```

### ۳. تنظیم لینوکس برای روشن کردن خودکار ربات بعد از ری‌استارت شدن سرور
```bash
pm2 startup
pm2 save
```

دستورات کاربردی مدیریت ربات در PM2:
### مشاهده وضعیت و لیست ربات‌های روشن
```bash
pm2 list
```

### مشاهده لاگ‌ها و گزارش کارکرد ربات در لحظه
```bash
pm2 logs tbr-bot
```

### خاموش کردن ربات
```bash
pm2 stop tbr-bot
```

### روشن کردن مجدد ربات
```bash
pm2 restart tbr-bot
```
